Application Bancaire Web - Pay My Buddy

Projet de fin de formation OpenClassrooms - Développeur d'application Java

Ce projet a été réalisé dans le cadre de ma formation OpenClassrooms. L'objectif était de concevoir et développer une application bancaire web complète permettant aux utilisateurs de transférer de l'argent et de gérer leurs relations financières en toute sécurité. Ce projet m'a permis d'acquérir des compétences solides en Java, Spring Boot, MySQL, et en architecture MVC.

Contexte

Dans un contexte fictif, j’ai travaillé en tant que développeur junior pour la start-up Pay My Buddy, spécialisée dans les solutions bancaires et financières.

L'application devait permettre :

  • La création d'un compte sécurisé.
  • L'ajout d'amis et la gestion des relations financières.
  • La réalisation de transferts d'argent simples et fiables.

Fonctionnalités principales

  • Création d'un utilisateur avec mot de passe sécurisé (hachage BCrypt).
  • Connexion et déconnexion sécurisées.
  • Ajout et gestion des relations entre utilisateurs.
  • Réalisation de transferts d'argent.
  • Suivi des transactions dans une base MySQL.
  • Interface utilisateur intuitive via Thymeleaf.

Pages clés

  • /signup → Inscription
  • /login → Connexion
  • /transfer → Transfert d'argent
  • /profile → Profil utilisateur
  • /addfriend → Gestion des relations

Architecture technique

Le projet suit une architecture MVC (Model-View-Controller) :

  • Controllers : gèrent les requêtes utilisateur.
  • Services : logique métier et gestion des transactions.
  • Repositories : accès à la base de données.
  • Entities : modélisation des tables MySQL.
  • Templates : pages HTML générées avec Thymeleaf.

Technologies utilisées

  • Java & Spring Boot
  • MySQL
  • BCrypt pour la sécurité
  • Jacoco & Surefire Reports pour les tests
  • Git & GitHub
  • IntelliJ IDEA

Base de données

La base de données projetpaymybuddy contient 3 tables principales :

Table Rôle
User Informations des utilisateurs (email, mot de passe, solde, etc.)
Transaction Historique des transferts d'argent
Connection Relations d'amitié entre utilisateurs

Compétences acquises

  • Développement full stack avec Java/Spring Boot.
  • Mise en place d'une application sécurisée et scalable.
  • Conception et gestion d'une base de données relationnelle.
  • Application du modèle MVC et bonnes pratiques de développement.
  • Réalisation de tests unitaires et d'intégration.
  • Utilisation de Git et GitHub pour le versioning.
